# Harmonic Data Systems (HDS)
> **Computation need not outpace human breathing.**

23년간의 사유를 담아, 기술이 인간의 리듬에 맞춰 함께 호흡하고 휴식하는 지속 가능한 데이터 아키텍처를 제안합니다.

## 📂 프로젝트 구조 (Repository Map)

이 저장소는 HDS의 철학과 기술적 명세를 체계적으로 담고 있습니다.

* **[`/manuscript`](./manuscript)**: HDS의 핵심 논문(`main.tex`)이 보관된 학술적 공간입니다.
* **[`/essays`](./essays)**: 기술을 넘어 대중과 소통하기 위한 쉬운 해설서들이 담겨 있습니다.
* **[`/spec`](./spec)**: 인간-AI 협약(`covenant.md`) 및 시스템의 상세 명세를 확인하실 수 있습니다.
* **[`/assets`](./assets)**: 프로젝트를 시각적으로 설명하는 자산들을 모아두는 곳입니다.
